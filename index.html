Simple Present Tense

Used for habits, facts, and regular actions

Structure: Subject + Verb (s/es) + Object
Example: "She reads books every day."
Negative: "She does not read books."
Question: "Does she read books?"

Simple Past Tense

Used for completed actions in the past

Structure: Subject + Verb (past form) + Object
Example: "I visited Jakarta last week."
Negative: "I did not visit Jakarta."
Question: "Did you visit Jakarta?"

Present Continuous Tense

Used for actions happening right now

Structure: Subject + is/am/are + Verb-ing
Example: "They are studying English now."
Negative: "They are not studying."
Question: "Are they studying?"

Variables & Data Types

JavaScript variables store data values.

JavaScript
// Variables
let name = "John";
const age = 25;
let isStudent = true;

// Arrays
let fruits = ["Apple", "Banana", "Orange"];

// Objects
let person = {
    name: "John",
    age: 25,
    city: "Jakarta"
};

console.log(person.name); // "John"

Functions

Functions are reusable blocks of code.

JavaScript
// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Arrow function
const add = (a, b) => a + b;

// Using functions
console.log(greet("World")); // "Hello, World!"
console.log(add(5, 3));      // 8

Loops & Conditions

Control flow in Python.

Python
# If-else condition
score = 85
if score >= 90:
    print("Excellent!")
elif score >= 70:
    print("Good job!")
else:
    print("Keep trying!")

# For loop
for i in range(5):
    print(f"Number: {i}")

# While loop
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

Linear Equations

Solve equations with one variable

2x + 5 = 15

Step 1: 2x = 15 - 5

Step 2: 2x = 10

Step 3: x = 10 / 2

Answer: x = 5 ✓

Quadratic Equations

Solve ax² + bx + c = 0

x² - 5x + 6 = 0

Method: Factoring

Step 1: (x - 2)(x - 3) = 0

Step 2: x - 2 = 0 or x - 3 = 0

Answer: x = 2 or x = 3 ✓

Area Formulas

Calculate areas of common shapes

▢ Rectangle: A = length × width

△ Triangle: A = ½ × base × height

○ Circle: A = π × r²

▱ Trapezoid: A = ½ × (a + b) × h

Pythagorean Theorem

For right triangles: a² + b² = c²

a² + b² = c²

Example: a = 3, b = 4, find c

Step 1: 3² + 4² = c²

Step 2: 9 + 16 = c²

Step 3: c² = 25

Answer: c = 5 ✓
